The goal is simple: Make a downloaded music player that is better, functionally and aesthetically, than the existing options.

Presenting the solution: **Mellifluent** _(sweetly or smoothly flowing; sweet-sounding)_

_Note: Mellifluent will never eat your files nor rummage through them for cat pictures, since as much as I may want your cat pictures, that goes against what is right, and similarly will never collect/send telemetry (it doesn't even get internet access)._

Installation
------------

1. Go to the releases tab and download the latest release for your OS, don't just hit the big green code button, the license and readme are all you will get.
https://github.com/DriftyCat/Mellifluent/releases/

3. Run the suspiciously small .exe file you just downloaded (it's almost like it has no bloat it is so small)

4. If it complains at you that you are downloading a virus, rest assured that I would never do that to you. The solution is to click '**More Info**', then a '**Run Anyway**' button will appear. You can put the downloaded file wherever (though I recommend putting it somewhere memorable so it doesn't get lost), it will take very little space and simply references your music files.

Usage
-----
You should now have the following screen
<img width="1918" height="1078" alt="image" src="https://github.com/user-attachments/assets/1edb3174-98af-4357-bc06-e1bf800d9d87" />

I recommend investigating the settings menu to change the theme to whatever your favorite is, or click the **+** besides **Folder** to add a new folder of music.
_The easiest way to do this is to have one core music file, as it will recursively search and update based on the contents_
